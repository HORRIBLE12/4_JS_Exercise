 # JavaScript Operators

 ## Article Section

 JavaScript can perform arithmetic operations.

**JavaScript can do math!**

Arithmetic operators are used to perform math between variables and/or values.

| Operator      | Description    | Example |
| ------------- | -------------  | ------- |
| +             | Addition       | x + y   |
| -             | Subtraction    | x - y   |
| *             | Multiplication | x * y   |
| /             | Division       | x / y   |
